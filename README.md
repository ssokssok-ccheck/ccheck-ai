# ccheck-ai

### 1. 저장소 클론

```bash
git clone https://github.com/ssokssok-ccheck/ccheck-ai.git
cd ccheck-ai
```

### 2. 가상환경 생성 및 실행

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. 라이브러리 설치

```bash
python3 -m pip install torch torchvision transformers pillow opencv-python pyserial requests
```

### 4. 메인 실행

```bash
python3 main_dual_arduino.py
```
